# 작업 계획 수립

사용자 요청: $ARGUMENTS

---

## 실행 순서

### Phase 1: 탐색
`explore-codebase` subagent를 사용하여:
- 프로젝트 구조 분석
- 관련 파일 식별
- 기존 패턴 파악
- 의존성 확인

### Phase 2: 계획
`plan-implementation` subagent를 사용하여:
- 상세 구현 계획 수립
- 작업 분해 및 의존성 정의
- 병렬 실행 가능한 작업 그룹화
- 리스크 분석 및 대응책

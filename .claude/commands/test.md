# 테스트 실행

대상: $ARGUMENTS

---

## 실행

### 1단계: 테스트 실행
`verify-code` subagent를 사용하여:
- 단위 테스트 실행
- 통합 테스트 실행 (있으면)
- 커버리지 확인 (가능하면)

### 2단계: 결과 분석

#### 모두 통과
→ 결과 요약 보고

#### 실패 있음
→ `debug-issues` subagent로 실패 원인 분석
→ 수정 방안 제시

---

## 출력

### 테스트 결과
- 총 테스트: [N개]
- 통과: [N개]
- 실패: [N개]
- 스킵: [N개]

### 실패 상세 (있으면)
[실패한 테스트별 원인과 수정 방안]

### 커버리지 (가능하면)
- 전체: [N%]
- 주요 파일별 커버리지

# 한투 퀀트 프로젝트 로드맵

## 현재 진행 상황

### 완료된 기능
- 프로젝트 기본 구조 설정
- API 연동 기본 프레임워크
- 모의투자 API 연동
- 모멘텀 전략 기본 구현
- 백테스트 엔진 기본 구조

### 진행 중인 기능
- API 인증 및 토큰 관리 안정화
- 실시간 시세 처리 개선
- 자동 매매 기능 테스트

## 우선 순위 작업

1. API 인증 및 토큰 관리 안정화
   - 모의투자/실제투자 환경 구분 명확화
   - 오류 처리 및 재시도 로직 개선

2. 백테스트 엔진 완성
   - 성과 지표 계산 기능 구현
   - 결과 시각화 및 리포트 생성

3. 전략 최적화 및 확장
   - 모멘텀 전략 매개변수 최적화
   - 추가 전략 구현 및 테스트
